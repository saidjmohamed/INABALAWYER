# إنابة ومعلومة - منصة المحامين

التطبيق الرائد لتبادل الإنابات والمعلومات بين المحامين في الجزائر

## الميزات الرئيسية

- PWA قابل للتنصيب
- مساعد ذكي بGoogle AI
- تواصل واتساب مدمج
- أمان عالي للبيانات

## إعداد المشروع

1. استنسخ المشروع
2. نصّب npm install
3. انسخ cp .env.example .env
4. شغّل npm run dev
